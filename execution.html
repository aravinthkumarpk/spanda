inherited from upstreamhard constraints (every fork item must respect)

  • Never hardcode state names. Always go through MemoryWorkflowDescriptor fields. Prefix-matching state.startsWith("ready_for_") silently breaks custom profiles.
  • Fail loudly. Missing config throws DispatchFailureError with the marker FOOLERY DISPATCH FAILURE. No silent fallbacks, no ?? "default".
  • Hermetic tests. src/**/__tests__/ = no fs / spawn / network / host binaries. Real-env tests go to __manual_tests__/ and never block CI.
  • Quality gates. Pre-push: bun run lint && test && tsc --noEmit && build. All four green or no push.
  • Size limits. 500 LOC/file · 100 LOC/function · 100 cols/line (ESLint enforced).
